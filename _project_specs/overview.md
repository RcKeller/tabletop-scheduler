# Project Overview

## Vision
A tabletop gaming session scheduler for coordinating game nights. Helps groups organize and schedule their tabletop RPG sessions.

## Goals
- [ ] Allow users to create and manage gaming groups
- [ ] Schedule recurring or one-time gaming sessions
- [ ] Track player availability
- [ ] Send reminders for upcoming sessions

## Non-Goals
- Not a virtual tabletop (no game mechanics)
- Not a character sheet manager
- Not a campaign wiki

## Success Metrics
- Users can successfully schedule a gaming session
- Group members receive notifications
- Easy to see upcoming sessions at a glance
